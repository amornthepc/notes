# The Box Drawing Digraphs (Neovim)

While in **Insert Mode**, press `<Ctrl-k>`, then type two letter code for the line you want:

- `│` (Vertical Line): press `<Ctrl-k>` then type `vv` (_vertical-vertical_)
- `─` (Horizontal Line): press `<Ctrl-k>` then type `hh` (_horizontal-horizontal_)
- `└` (Bottom-left Corner): press `<Ctrl-k>` then type `ur` (_up-right_)
- `├` (T-junction split): press `<Ctrl-k>` then type `vr` (_vertical-right_)
