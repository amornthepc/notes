# Current Working Directory

**Current Working Directory (`.`)** is the directory that you are currently active in the filesystem. — your current location in the terminal.

`.` is a special alias that refers to "**the current working directory**". It's a shortcut that you can use when want to run a command on the directory you are currently in.

```bash
git add .
```

## Find Current Working Directory

You can find your current working directory by running the [`pwd` command](./pwd-command.md)

```bash
pwd
# /home/amornthep
```
