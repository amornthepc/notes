# Absolute Path

**Absolute Path** is a full path that _starts at the root_ of the filesystem on Unix-like systems.

> [!NOTE]
> **Remember**: the root is denoted by a forward slash `/`

## Absolute Path Example

```
root
 └── vehicles
      ├── cars
      │   ├── fords
      │   │   ├── mustang.txt
      │   │   └── focus.txt
```

No matter where your current working directory is, The absolute path to the `mustang.txt` always the same:

```
/vehicles/cars/fords/mustang.txt
```
