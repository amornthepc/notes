# CLI vs GUI

- **CLI** is best for developer — Offers more control, easy automation, faster execution, look cooler.
- **GUI** is best for general end-user — Visual and easier to use.

## CLI (Command Line Interface)

**CLI** is a text-based way to interact with a computer by typing text commands to make the computer do things.

```bash
mkdir project/
```

## GUI (Graphic User Interface)

**GUI** is a visual way to interact with a computer by doing actions like clicking buttons, right-clicking to create a directory.

> [!NOTE]
> CLI and GUI are not actual programs — they're just the ways you use to talk with computer. For example Git can be used CLI way by typing `git` commands in terminal or GUI way by clicking buttons in an application like GitHub Desktop
