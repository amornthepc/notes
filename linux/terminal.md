# Terminal

**Terminal** is a desktop app that responsible for **accepting text commands (`stdin`)**, passing them to the _shell_, and **rendering/drawing results on the screen (`stdout`, `stderr`)**.

> [!NOTE]
> Nowadays, _Terminal_ is really mean _Terminal Emulator_ — a modern desktop app (like Kitty, Windows Terminal, etc) simulating legacy physical terminal hardware.
