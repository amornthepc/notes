# CLI (Command Line Interface)

**CLI** (**Command-Line (_Text_) Interface**) is a text-based interface use to interact with a computer system by typing text commands to make the computer do the tasks we want. — instead of clicking button in a graphic interface (GUI).

## CLI Command Syntax

Most command typed into the terminal follow a structured pattern:

```bash
<COMMAND> [OPTION]... [ARGUMENT]...
```

- `<COMMAND>`: The executable program you want to run (e.g `mkdir`, `ls`)
- `[OPTION]`: The optional modifier flags that change how command behave (e.g `-l`, `-r`)
- `[ARGUMENT]`: The data that the command uses during execution, Depending on the command, this can be required or optional, such as file paths, directory names, or text strings.

> [!NOTE]
> CLI is the best workflow for developer, It offers absolute more control, make automation easy, run incredibly fast, and looks cool af.
