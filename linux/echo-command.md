# `echo` command

`echo` command used to print specified text or variables to standard output (`stdout`)

```bash
echo [OPTION]... [STRING]...
```

## Practical Reference:

- **Print Text**:

```bash
echo "I use Linux btw"
# Output: I use Linux btw
```

- **Print Variable**:

```bash
USER="Hangman"
echo $USER
# Output: Hangman
```

- **Write to file (replace)**:

```bash
echo "text" > file.txt
```

- **Write to file (append)**:

```bash
echo "text" >> file.txt
```
