# Terminal Shortcut

There are special keyboard inputs used to send hardware signals, control processes and navigate text in the terminal.

## Core Signals (CTRL KEY)

- **`Ctrl + d` (`^d`)**: **DONE** sends the **EOF** (End of File) signal to closes the [`stdin`](./standard-input.md) cleanly.
- **`Ctrl + c` (`^c`)**: **CANCEL** sends the **SIGINT** (Interrupt) signal to kills the currently running process immediately.

## Navigation (ALT KEY)

- `Alt + f` (`M-f`): Move cursor **forward** by one word.
- `Alt + b` (`M-b`): Move cursor **backward** by one word.

## My Personal Configurations

- `Alt + n`: Create **New** tab in terminal.
- `Alt + x`: **Close** current tab.
- `Alt + r`: **Rename** tab.
- `Alt + h`: Move to **previous** tab.
- `Alt + l`: Move to **next** tab.
