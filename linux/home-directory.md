# Home Directory (`~`)

**Home Directory** is a _user's private directory_ where their personal files are stored. — It is also the default directory where a user starts when logging in the system or opening a new terminal.

The home directory path is different depending on operating system:

- _macOS_: Mostly located at `/Users/<username>`
- _Linux_: Mostly located at `/home/<username>`.

> [!NOTE]
> **Highly Recommended:** do all your coding work inside your home directory, For example create `workspace/` directory inside your home directory, and keep all your programming projects organized in sub-directories there.

## The `~` Alias

`~` (_tilde_) character is an alias (**a shortcut**) for _home directory_

```bash
# Both of these commands take you to the exact same place which is home
cd ~
# or
cd /home/amornthep
```
