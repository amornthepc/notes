# Prompt vs Output

```bash
amornthepc:~$ echo Hello
Hello
amornthepc:~$
```

- **Prompt**: A text string (`amornthepc:~$`) that tells the system is ready for input.
- **Command**: The text-based command that you type into terminal (e.g. `echo`), additional words following are arguments. (e.g. `Hello`)
- **Output** is _text_ data `Hello` (`stdout`, `stderr`) sent back by the program through the shell.
