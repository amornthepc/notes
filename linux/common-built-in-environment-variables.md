# Common Built-in Environment Variables

Most use built-in [environmet variables](./environment-variables.md).

| Variables | What it Stores                                                                    | Example                     |
| --------- | --------------------------------------------------------------------------------- | --------------------------- |
| USER      | Account username                                                                  | `amornthep`                 |
| HOME      | Absolute path to your [home directory](./home-directory.md)                       | `/home/amornthep`           |
| PWD       | Absolute path of your [current working directory](./current-working-directory.md) | `/home/amornthep/workspace` |
| SHELL     | Path to your active shell program                                                 | `/bin/bash`                 |
| PATH      | List of directories used to search for commands                                   | `/usr/local/bin:/usr/bin`   |
