# Parent Directory

**Parent Directory** is a directory that contains other directories (1 level up) — the directories inside it are called _subdirectory_ or _child directory_

`..` is a special alias that refers to **"the parent directory"**. It is use as shortcut to refer to the directory above your current location.

You can chain multiple `..` path together to step back even further.

```bash
cd ..
# Move up one level in the directory tree

cd ../..
# Move up two level in the directory tree
```
