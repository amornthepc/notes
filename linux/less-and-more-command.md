# `less` and `more`

`less` and `more` command are used to view the contents of a file in interactive mode.

They take over the terminal window so you can interact with them to scroll through a file one line or one page at a time. You can press `q` to quit back to your shell prompt.

> [!NOTE]
> As a general rule, Use `less` instead of `more`. It does the same job but with more feature. (like allowing scroll backward.)

```bash
less file1.txt
```
