# `history` command

`history` command is used to display a list of previously executed commands that you've typed in your shell.

**Syntax**: **Only command**, no option, no argument

```bash
history
```

```bash
history
# 1 cd project
# 2 ls
# 3 history
```

## Navigate History (Up and Down Arrow Keys)

You can use **Up and Down Arrow** key to quickly cycle through your command history back and forth directly from your prompt.

- Press the **Up Arrow** key to go _backward_ through the command history, (_previous_ command).
- Press the **Down Arrow** key to go _forward_ through the command history, (_next_ command).
