# Operating System (OS)

**Operating System (OS)** is the foundational collection of software that makes a computer usable by managing system resources and acts as the interface/bridge between user, applications, and the machine.

## What It Includes

- **User Interface (UI)**: The visual interactive layer that allow you to control the computer (GUI or CLI).
- **System Utilities**: The essential built-in tools required to manage the system (file manager, configuration tools, text-editor)
- **Kernel**: The core engine of the OS. It is the _only_ software that allowed to directly communicate and control the physical hardware.

## Primary Purposes

- **Abstraction**: Hide the complex messy details of physical hardware behind a software interface. (So users and programs can interact with the computer easily without needing to write or understanding low-level hardware code).
- **Resource Management**: Fairly allocates CPU time, RAM, and storage space across all running applications. (_traffic cope_)
- **Security & Isolation**: Prevent one misbehave app from crashing the entire system or stealing data from another app. (_sandboxes processes_)

## Hardware vs. OS

- **Hardware**: The physical components that you can physically touch (CPU, RAM, SSD).
- **Operating System**: The software code stored on the drive that wakes up and commands the hardware.

## Examples

- Desktop/Server: Linux, macOS, Windows
- Mobile: Android, iOS
