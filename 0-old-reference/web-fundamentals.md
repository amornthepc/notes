# Web Fundamentals

## Network

**Network** is a group of computers/devices that connected together so they can communicate to each other.

## Internet

**Internet** is the _global network_ (network of networks) that connected all computers/devices around the world together so the whole world can communicate.

## IP Address (Internet Protocol)

**IP** is a _unique address for device that connected to network_, allowing data sent to the right location. IP made of four numbers separated by dots. (example: `192.0.2.172`, `192.168.1.1`)

## Router

**Router** is a computer that _connects different networks together_ and _forwards data packet to the right recipient address_ directly.

## ISP (Internet Service Provider)

**ISP** is a company (like TRUE, AIS) that manages special _routers_ which all connected together and can access other ISPs' routers so it _provides access to the global internet_.

## Packets

**Packets** is _a small chunk of data sent across a network._ (between client & server) instead of sending one large data, file or web page, _the system breaks it down into multiple tiny units for transit_ then once all the packets arrive _at destination the computer reassembles them back together_ to recreate the original data.

### Packet Components

- **Header**: contains metadata for delivery to tell the computer where packet came from and where it's going -> Which include server and client IP address, packet number, total number of packets and protocol used in the transmission.
- **Payload**: contains the actual data sent in the packet.

## Client

**Client** is a device or software application (such as laptop or browser) that request data or service from server. (not connect to internet directly access via ISP)

## Server

**Server** is a computer that store software program (like web app) waits for incoming requests from clients and responds by providing data or service.

## Web Page

**Web Page** is a document typically written in HTML, designed to be viewed in web browser. Often called just "pages"

## Website

**Website** is a collection of webpages grouped together into a single resource with links connecting them together. Often called just "site"

## Web Server

**Web Server** is **computer that hosts a website on the Internet** (store files like HTML, CSS, JavaScript and images) and _serves_ clients who request them.

## Web Service

**Web Service** is a software program that responds to requests over the internet to perform a specific function or provide data (e.g. handling user login, processing payment)

## Search Engine

**Search Engine** is a type of web service that help you find other web pages on the internet (such as Google, Yahoo, DuckDuckGo), Search engines are accessed via web browser

## DNS request (DNS = Domain Name System)

**DNS request** is process of translating a domain name (human-readable like google.com, youtube.com) into its corresponding IP address (machine-readable), so the computer know exactly where to send request to.

## Web Browser

**Web Browser** is software application installed on your devices (computer, phone) to access, retrieve and view website.
